# Motor-detection
Motor-detection，一个电机运行时的检测系统，可以实时监测电机的运行状态，统计运行数据，支持分析，导出，使用方便，值得一试。


------------------------------
## 快速开始

### Windows
下载最新的固件烧录进ESP32，连接上其AP，WIFI默认SSID为**Motor-detection**;也可以下载MCU源码，自行编译。
常见问题请查阅：https://docs.hardwork.icu
